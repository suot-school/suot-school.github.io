Задание 1

У тебя есть список участников розыгрыша:

uchastniki = ['Маша', 'Лена', 'Федя', 'Женя', 'Даша']

Напиши программу, которая случайным образом выберет победителя из списка участников.
Попробуй запустить свой код несколько раз.

Trinket

Пример решения:

from random import choices
uchastniki = ['Маша', 'Лена', 'Федя', 'Женя', 'Даша']
result = choices(uchastniki)
print(result)
