Задание 1

У тебя есть список покупок, который оформлен как словарь:

pokupka = {'леденцы': 200, 'молоко': 80, 'булочки': 300}

Напиши программу, которая посчитает общую стоимость покупки.

Trinket

Пример решения:

pokupka = {'леденцы': 200, 'молоко': 80, 'булочки': 300}
spisok = pokupka.values()
result = sum(spisok)
print(result)
